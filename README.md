Laravel 5.6 Library for providing authentication facility to RwPay build to use CoreUser

It provide feature like facebook login or google login but with Tekana app

Requirements
============

* PHP >= 7.2.0
* cURL Extension

Installation
============

    composer require tekana/tekanaauth:dev-master
